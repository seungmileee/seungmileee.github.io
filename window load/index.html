// index.html <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8" /> <meta http-equiv="X-UA-Compatible" content="IE=edge" /> <meta name="viewport" content="width=device-width, initial-scale=1.0" /> <title>window load</title> <style> div { display: inline-block; border: 3px solid pink; border-radius: 10px; padding: 5px; } </style> </head> <body> <div>window load 공부해보기</div> <script> window.addEventListener('load', () => console.log('load')) window.addEventListener('DOMContentLoaded', () => console.log('DOMContentLoaded'), ) </script> </body> <script src="script.js" defer></script> </html> // script.js console.log('js loaded -> defer')

load : html, js, css, 이미지 등등의 모든 리소스가 다 로드되었을 때 이벤트 발생

DOMContentLoaded : html이 모두 로드되고 DOM 트리가 완성되었을 때 (외부 리소스 (이미지..)가 아직 로드 되지 않았을 때) 이벤트 발생

defer : html이 모두 로드되고 DOMContentLoaded 이벤트가 발생하기 전에 먼저 호출

이와 같이 defer -> DOMContentLoaded -> load 순으로 이벤트가 발생한다는 것을 확인할 수 있다.
